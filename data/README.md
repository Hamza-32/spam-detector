# Dataset Instructions

Download the UCI SMS Spam Collection dataset:

1. Open https://archive.ics.uci.edu/dataset/228/sms+spam+collection
2. Download and extract the archive.
3. Copy the `SMSSpamCollection` file into this folder.

Expected file path:

- `data/SMSSpamCollection`

Note:

- This repository keeps data files out of git history by default.
